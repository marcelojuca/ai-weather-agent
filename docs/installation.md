# Installation

## Stable release

To install AI Weather Agent, run this command in your terminal:

```sh
uv add ai-wheather-agent
```

Or if you prefer to use `pip`:

```sh
pip install ai-wheather-agent
```

## From source

The source files for AI Weather Agent can be downloaded from the [Github repo](https://github.com/marcelojuca/ai_wheather_agent).

You can either clone the public repository:

```sh
git clone git://github.com/marcelojuca/ai_wheather_agent
```

Or download the [tarball](https://github.com/marcelojuca/ai_wheather_agent/tarball/master):

```sh
curl -OJL https://github.com/marcelojuca/ai_wheather_agent/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ai_wheather_agent
uv pip install .
```
